# CST 438 – Project 3 Frontend

---

## Overview
This repository contains the frontend application for our CST 438 project.  
It is responsible for user interaction, UI/UX, and communicating with the backend API.

> This is part of an **early-stage development setup** and will evolve over time.

---

## Core Responsibilities
- Provide user interface and experience  
- Communicate with backend API  
- Handle user input and display data  
- Maintain consistent design across platforms  

---

## Tech Stack
- Reactive web framework  
- Hosted on **Render**  
- Mobile support (Android or iOS)  

---

## Features
- User authentication flow (OAuth2)  
- Account creation and management UI  
- Views/pages for core application features  
- Display and interaction with backend data  

---

## API Integration
- Connects to Spring Boot backend  
- Handles API requests and responses  
- Displays real-time data to users  

---

## Testing
- Jest for unit testing  
- Playwright for end-to-end testing  

---

## Deployment
- Hosted online (Render or similar)  
- Connected to GitHub for automatic deployment  
- Works with backend deployed separately  

---

## Notes
> Frontend should remain clean, responsive, and user-focused  
> UI and features may evolve during development
